Edibit JS Practice: Return The Last Item In An Array

Create a function that accepts an array and returns the last item in the array. The array can contain any of JavaScript's five primitive data types.

  1. Number
  2. String
  3. Boolean
  4. Undefined
  5. Null

Examples

  • [1, 2, 3] => 3
  • ['cat', 'dog', 'duck'] => 'duck'
  • [true, false, true] => true

Gotchas

No gotchas. All test cases contain valid data types.
